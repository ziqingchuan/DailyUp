# DailyUp

一个简洁美观的工作日报/周报管理应用，支持心情记录、个性化卡片生成和数据可视化。

## 功能特性

- ✍️ 记录日报和周报
- 😊 心情记录与趋势分析
- 🎨 生成报告卡片
- 📤 导出功能（JSON 格式和图片格式）
- 📥 导入功能（支持同格式 JSON 文件）
- 📊 数据可视化（ECharts 心情趋势图和工作雷达图）
- 📈 统计总结和趋势分析
- 🔍 搜索和筛选报告
- 💾 本地存储，无需后端
- 📱 完整的移动端适配

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 技术栈

- React 18
- Vite
- ECharts（数据可视化）
- HTML2Canvas（卡片导出）
- date-fns（日期处理）
- LocalStorage（数据存储）
- 响应式设计（移动端适配）

## 使用说明

1. 点击"新建报告"创建日报或周报
2. 填写工作内容、收获、计划和心情等信息
3. 在报告列表中查看、编辑或删除报告
4. 点击"查看卡片"生成精美的可视化卡片
5. 使用"导出图片"功能保存卡片
6. 在"统计总结"页面查看数据分析和心情趋势
